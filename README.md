# Vibe Apps

A monorepo for quick, vibe-coded applications.

## Apps

### Multi-Store Search
Price comparison app that searches Amazon, Blinkit, and Zepto simultaneously in one view.

**Location:** `packages/multi-store-search`

**Run:** `npm run price-compare`

## Setup

```bash
npm install
```

## Adding New Apps

Create a new package under `packages/` and add a script in the root `package.json`.
