# Multi-Store Search

Electron app for comparing prices across Amazon, Blinkit, and Zepto simultaneously.

## Features

- Search all three stores at once from a single search box
- View all stores side-by-side in one window
- Auto-set location to Hosur, TN
- Open any store in external browser at current state
- Persistent sessions for each store

## Run

```bash
npm install
npm start
```

## Tech Stack

- Electron
- Webviews for embedding store sites
